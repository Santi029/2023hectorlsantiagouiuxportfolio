Hi, I'm Hector L. Santiago

My name is Hector L. Santiago. I work as a UI/UX Designer for Copart International located in Dallas, Texas.

Currently, I’m designing the next generation of the Copart Website for US and two other subsidaries called CashforCars.com and CrashedToys.com with two of the best team in the company. Previously I was working as a Lead Manager for Amazon, Web advisor for Hostgator and many years as a web designer and developer in San Juan, Puerto Rico.

I am a dedicated and passionate UI / UX designer from Frisco, Texas. Since I started my learning journey as a web designer and web developer in 2008, I have enjoyed how design is created and transformed into a great website or app. I've always found it interesting how to create a layout from a user-centric approach and then bring it into code achieving the preset layout.

I have years of experience working with the Adobe Master Collection since version 3, along with programming languages such as HTML, CSS, and JavaScript. The programs I use the most are Visual Studio Code, Git, GitHub, Illustrator, XD and recently Figma.

I proudly consider myself a lifelong learner in this industry because technology never stops.
